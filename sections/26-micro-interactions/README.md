## Quality Constraints

1. **Response time**: < 100ms
2. **Consistency**: Same feel everywhere
3. **Delightful**: Not distracting

---

## Interaction Inventory

### Buttons
- Hover: scale(1.02), shadow, glow
- Active: scale(0.98)
- Focus: 2px ring

### Inputs
- Focus: lift, glow, border
- Valid: checkmark
- Invalid: shake, X

### Cards
- Hover: lift, shadow
- Active: slight press

### Links
- Hover: underline draws
- Active: color shift

### Scrollbar (optional)
- Custom styled
- Matches theme

### Cursor (optional)
- Custom cursor
- Changes on interactive elements
